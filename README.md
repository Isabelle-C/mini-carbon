# mini-carbon
Python CLI tool that minimally recreates Carbon (https://carbon.now.sh)

## Environment
- Managed with [Poetry](https://python-poetry.org/)
```bash
poetry install
```

## Usage
1. Activate environment
```bash
poetry shell
```

2. Run the script
```bash
python cli.py input_example.txt python output_example.png
```
